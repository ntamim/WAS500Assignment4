Book Details

Learn to Code HTML & CSS

Learn to Code HTML & CSS is a beginner's guide to the basics of web development and designing websites. This book covers the latest version of HTML5 and CSS3, with new chapters on creating responsive and accessible websites. You'll learn how to write semantic code that not only works in multiple browsers, but is also easy for other developers to read and maintain. You'll also learn how to use CSS3 to style your websites, including how to use the latest CSS3 features such as flexbox and CSS Grid.
